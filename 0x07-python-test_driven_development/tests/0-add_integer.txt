This library is very simple, since it only has one function called
"add_integer()"
--------------------------------
First import "add_integer" from the "0-add_integer" module:

::

    >>> add_integer = __import__('0-add_integer').add_integer

Function that adds 2 integers, Returns an integer:
the addition of a and b, a and b must be integers or floats.

::

    >>> add_integer(8, 16)

a must be integer or float, otherwise raise a TypeError exception
with the message a must be an integer

::

    >>> add_integer('x', 16)
    Traceback (most recent call last):
    TypeError: a must be an integer

b must be integer or float, otherwise raise a TypeError exception
with the message b must be an integer

::

    >>> add_integer(8, 'x')

    TypeError: b must be an integer
